# FDB-Applet
FDB Applet is developed as a computational tool to facilitate research on problems arising in the representation theory of gentle, string (and ultimately biserial) algebras. In the development of the FDB Applet, we benefited substantially from the valuable FD Applet released in 2023 by Haruhisa Enomoto. Thus, FDB is viewed as the "B" version of FD.
